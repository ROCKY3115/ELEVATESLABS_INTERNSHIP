# 📊 Task 4 – Dashboard Design | Data Analyst Internship

## 📌 Overview

This repository contains my submission for **Task 4 – Dashboard Design** as part of the **Data Analyst Internship at Elevate Labs**.

The objective of this task was to design an interactive Power BI dashboard that enables business stakeholders to analyze product performance, pricing, customer ratings, and sales-related insights for better decision-making.

---

## 🎯 Objective

- Design an interactive dashboard using **Power BI**
- Create meaningful KPIs and visualizations
- Implement filters (slicers) for interactive analysis
- Present insights that support business decisions

---

## 🛠️ Tools Used

- Microsoft Power BI
- Power Query
- DAX (Basic Measures)
- CSV Dataset

---

## 📂 Dataset

The dashboard is built using an Amazon product dataset containing information such as:

- Product ID
- Product Name
- Category
- Actual Price
- Discounted Price
- Discount Percentage
- Product Rating
- Rating Count
- Review Title
- Review Content

---

## 📈 Dashboard Features

### 📊 Visualizations

- **Bar Chart:** Average Discount by Category
- **Scatter Plot:** Rating vs Rating Count (Popularity vs Quality)
- **Bar Chart:** Category-wise Product Count
- **Donut Chart:** Actual Price Distribution by Category
- **Table:** Product Summary with Price, Rating & Reviews
- **Column/Line Chart:** Discounted Price Analysis by Customer Reviews

### 🎛 Interactive Slicers

- Actual Price Range
- Product Category

---

## 📌 Key Insights

- Electronics contribute the highest share of product value.
- Some products receive a large number of reviews despite having average ratings.
- Certain product categories consistently offer higher discounts.
- Product pricing and customer feedback provide valuable insights into purchasing trends.

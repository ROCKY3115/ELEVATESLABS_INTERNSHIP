# 🧹 TASK_1 – Data Cleaning & Preprocessing

## 🎯 Objective
Clean and preprocess a raw customer dataset by handling missing values, removing duplicates, correcting data inconsistencies, and preparing the data for analysis.

## 🛠️ Tools Used
- Python
- Pandas
- Jupyter Notebook

## 📂 Dataset
- Customer Personality Analysis Dataset

## ✅ Steps Performed
- Loaded the dataset using Pandas.
- Checked for missing values and data types.
- Removed unnecessary columns.
- Filled missing values in the **Profession** column with **"Unknown"**.
- Removed duplicate records.
- Encoded categorical columns using Label Encoding.
- Exported the cleaned dataset as **Cleaned_Customers.csv**.

## 📁 Files Included
- Customers.csv
- Cleaned_Customers.csv
- DataCleaning_.ipynb
- README.md

## 📌 Outcome
Successfully cleaned and prepared the dataset for further analysis by improving data quality and consistency.

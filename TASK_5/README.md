# 🚢 Task 5 – Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Objective

This project is submitted as **Task 5** for the **Data Analyst Internship at Elevate Labs**.

The objective is to perform **Exploratory Data Analysis (EDA)** on the Titanic dataset using Python to uncover patterns, trends, correlations, and insights through statistical analysis and visualizations.

- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Project Files

- `README.md` – Project documentation

---

## 📊 Exploratory Analysis Performed

- Dataset overview using `.info()` and `.describe()`
- Missing value analysis
- Survival count visualization
- Age distribution analysis
- Survival by gender
- Age vs Passenger Class
- Correlation Heatmap
- Summary of findings

---

## 📈 Key Insights

- Female passengers had significantly higher survival rates than male passengers.
- First-class passengers had the highest probability of survival.
- Higher ticket fares were positively associated with survival.
- Most passengers were between 20 and 40 years of age.
- Third-class passengers were generally younger than first-class passengers.
- Missing values were found in the **Age**, **Cabin**, and **Embarked** columns.

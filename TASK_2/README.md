
# 📊 TASK_2 – Data Visualization & Storytelling

## 🎯 Objective
The objective of this task was to create an interactive dashboard that transforms raw sales data into meaningful business insights using effective data visualization techniques.

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- Superstore Sales Dataset (CSV)

## 📂 Dataset
- Sample - Superstore.csv

## 📈 Dashboard Visualizations
- Sales by Category (Bar Chart)
- Sales by Region (Pie Chart)
- Sales by Sub-Category (Treemap)
- Sales Trend by Order Date (Line Chart)
- Sales by Segment and Category (Stacked Bar Chart)
- Profit vs Discount vs Sales (Column Chart)

## 💡 Key Insights
- Technology is the highest-selling product category.
- The West region contributes the highest overall sales.
- Phones and Chairs are among the best-performing sub-categories.
- Consumer and Corporate segments generate strong revenue.
